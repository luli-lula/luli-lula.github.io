在线工具箱

项目简介 在线工具箱是一个集成了多种实用工具的Web平台，旨在为开发者和普通用户提供便捷的在线工具服务。无需安装软件，打开浏览器即可使用各种实用工具。 技术栈 前端框架: Vue.js 3 UI组件库: Element Plus 构建工具: Vite 后端框架: Node.js + Express 部署: Docker + Nginx 版本控制: Git 已实现工具 🔧 开发工具 JSON格式化: 美化和压缩JSON数据 Base64编解码: 文本和图片的Base64转换 URL编解码: URL参数的编码和解码 MD5加密: 文本MD5哈希生成 UUID生成器: 生成各种版本的UUID 🎨 设计工具 颜色选择器: RGB、HEX、HSL颜色转换 二维码生成: 支持文本、URL、WiFi等类型 图片压缩: 在线压缩图片大小 Icon制作: SVG图标生成和编辑 📝 文本工具 文本差异对比: 可视化文本对比 字数统计: 统计字符、单词、段落数量 正则表达式测试: 在线正则匹配测试 Markdown编辑器: 实时预览的Markdown编辑器 🧮 实用工具 时间戳转换: Unix时间戳与日期互转 单位换算: 长度、重量、温度等单位转换 密码生成器: 生成安全密码 IP地址查询: 查询IP归属地信息 项目特色 🚀 高性能 前端采用Vue 3 Composition API，响应速度快 使用Vite构建，开发体验优秀 静态资源CDN加速 🎯 用户友好 简洁直观的界面设计 响应式布局，支持移动端 深色模式支持 工具分类清晰 🔒 隐私保护 大部分工具在前端处理，数据不上传服务器 支持离线使用 不收集用户个人信息 架构设计 前端架构 src/ ├── components/ # 公共组件 │ ├── Layout/ # 布局组件 │ ├── Tools/ # 工具组件 │ └── Common/ # 通用组件 ├── views/ # 页面视图 ├── router/ # 路由配置 ├── store/ # 状态管理 ├── utils/ # 工具函数 └── styles/ # 样式文件 后端架构 server/ ├── controllers/ # 控制器 ├── routes/ # 路由定义 ├── middleware/ # 中间件 ├── services/ # 业务逻辑 └── utils/ # 工具函数 开发亮点 1. 组件化设计 每个工具都设计为独立的Vue组件，便于维护和扩展： ...
