Why#

CELPIP（Canadian English Language Proficiency Index Program）是加拿大政府官方认可的英语水平测试，对于想要移民加拿大的人来说至关重要。然而，市面上缺乏优质的CELPIP写作练习工具，大多数学习者只能依靠昂贵的培训课程或者缺乏反馈的自主练习。

作为一名技术开发者，我看到了AI技术在语言学习领域的巨大潜力，决定构建一个智能化的CELPIP写作练习平台，让学习者能够获得即时、专业的写作反馈。

What#

CELPIP12 是一个全功能的AI驱动写作练习平台，提供：

🤖 智能AI评分系统#

  • 基于OpenAI o1-mini模型的专业评分
  • 严格按照CELPIP官方评分标准（1-12分制）
  • 四维度评估：内容连贯性、词汇运用、可读性、任务完成度

📝 实时写作反馈#

  • 逐词级别的语法和表达建议
  • 智能高亮显示需要改进的部分
  • 详细的改进建议和解释

📚 官方范文对比#

  • 收录90+篇官方范文，覆盖不同分数段
  • 智能检测题目来源，自动匹配相关范文
  • 分数段筛选，学习不同水平的写作技巧

🎯 专业题库系统#

  • 46+道官方练习题目
  • Task A（邮件写作）和Task B（调查回应）完整覆盖
  • 智能排序和搜索功能

👤 用户管理系统#

  • Clerk集成的安全认证系统
  • 个性化学习进度跟踪
  • 写作历史记录和成绩分析

How#

技术架构#

前端技术栈#

  • Next.js 14: 基于App Router的现代React框架
  • TypeScript: 全类型安全的开发体验
  • Tailwind CSS: 响应式UI设计
  • Slate.js: 富文本编辑器，支持实时修改建议高亮

后端与服务#

  • API Routes: Next.js服务端API，处理AI评估和数据管理
  • OpenAI Integration: o1-mini模型API集成，专业的语言评估
  • Vercel Postgres: 云端数据库，存储用户数据和学习记录
  • Clerk Authentication: 企业级用户认证和管理

核心算法设计#

双阶段AI评估系统：

// 第一阶段：综合评分
const scoreResponse = await openai.chat.completions.create({
  model: "o1-mini",
  messages: [{
    role: "user", 
    content: celpipTeacherPrompt + userEssay
  }]
});

// 第二阶段：详细修改建议  
const correctionResponse = await openai.chat.completions.create({
  model: "o1-mini",
  messages: [{
    role: "user",
    content: celpipCommentsPrompt + userEssay
  }]
});

智能缓存机制：

  • 全局积分状态管理，避免重复API调用
  • 用户初始化自动化，提升新用户体验
  • 30秒缓存机制，平衡性能与数据新鲜度

系统设计亮点#

1. 用户体验优化#

  • 主动初始化架构: 用户登录后自动完成账户初始化，消除错误提示
  • 实时保存: 自动保存写作内容，防止意外丢失
  • 响应式设计: 适配各种设备，提供一致的学习体验

2. 数据架构设计#

-- 用户表
CREATE TABLE users (

  email TEXT,
  usage_count INTEGER DEFAULT 30

-- 评分历史表  
CREATE TABLE score_history (
  id SERIAL PRIMARY KEY,
  user_id TEXT,

  essay TEXT,
  ai_response JSONB,

3. 错误处理与重试机制#

  • 指数退避重试策略，确保AI API调用的稳定性
  • 优雅的错误恢复，提升用户体验
  • 详细的错误日志，便于问题诊断

技术挑战与解决方案#

挑战1: AI评分的准确性和一致性#

解决方案:

  • 精心设计的prompt工程，确保AI输出符合CELPIP评分标准
  • 多轮测试和调优，保证评分的稳定性
  • JSON格式化输出，确保数据结构的一致性

挑战2: 复杂的用户状态管理#

解决方案:

  • Context-based全局状态管理
  • 单一数据源架构，避免状态不同步
  • 智能缓存策略，减少不必要的API调用

挑战3: 富文本编辑器的修改建议高亮#

解决方案:

  • Slate.js自定义渲染器
  • 基于字符索引的精确定位
  • 动态样式应用，实现实时高亮效果

核心功能实现#

智能评分引擎#

基于OpenAI o1-mini模型，我设计了双阶段评估系统：

  1. 整体评分: 根据CELPIP官方评分标准提供1-12分的综合评价
  2. 细节修改: 逐词分析，提供具体的改进建议和语法修正

范文智能匹配#

  • 自动识别用户选择的题目
  • 从90+官方范文中匹配相关内容
  • 支持按分数段筛选，学习不同水平的写作技巧

个性化学习路径#

  • 记录用户的写作历史和成绩变化
  • 提供个性化的学习建议
  • 支持多语言界面（中英文切换）

部署与运维#

生产环境部署#

  • Vercel: 零配置部署，自动CI/CD
  • 边缘计算: 全球CDN加速，降低访问延迟
  • 数据库: Vercel Postgres，自动扩缩容
  • 监控: 内置错误追踪和性能监控

安全性保障#

  • Clerk认证: 企业级用户认证和权限管理
  • API安全: 所有敏感操作都需要用户认证
  • 数据加密: 用户数据传输和存储全程加密
